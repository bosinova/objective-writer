# Objective Writer

Single-page React app for drafting Bloom-aligned learning objectives.

## Run locally

1. Install **Node.js (LTS)** from `https://nodejs.org/`
2. In this folder:

```bash
npm install
npm run dev
```

Then open the URL shown in your terminal (typically `http://localhost:5173`).

## What’s included

- Large textarea for raw course content
- Bloom’s Taxonomy dropdown: Remembering → Creating
- Audience description field
- Number of objectives input
- Generate button (placeholder now; ready for Claude API wiring later)
